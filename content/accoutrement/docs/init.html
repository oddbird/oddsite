QuickStart: Init

Light-weight browser normalization based (loosely) on Sanitize and Normalize.

Import

WARNING: This module writes CSS directly to your stylesheets. There is no configuration, no mixins, no variables, and no functions. This module is not imported by default, and does not require the core.

  @import '<path-to>/accoutrement/sass/init';

If you’re using Eyeglass, you can leave off most of the path:

  @import 'accoutrement/init';
