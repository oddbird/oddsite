*, ::before, ::after

scss 
*,
::before,
::after {
  border-style: solid;
  border-width: 0;
  box-sizing: border-box;
}

Specify the border style, width, and box-sizing of all elements.

  font-size: inherit;
  line-height: inherit;
  margin: 0;
  padding: 0;
}

Specify the core styles of all elements.
