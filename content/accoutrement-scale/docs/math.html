Math Functions

Use these internal math functions to adjust relative sizes in your map.

@function plus()

aliased as add()

Add two values together in a $sizes map.

Parameters & Return

$size1: (string | length)

The name or length of the size you are adding to

$size2: (string | length)

The name or length of the size being added

@return (number)

The calculated results of adding $size1 and $size2

Example

scss 
$sizes: (
  'text': 16px,
  'margin': 14px,
  'spacer': 'text' ('plus': 'margin'),
);

Requires

@function size()

Used By

@function add()
