Using WebFonts

Once you’ve configured your webfont data into a map, these tools will help you access the data in various ways: from creating @font-face imports, to calling font-families by any given alias.

@function font-url()

This is a wrapper for the CSS url() function, which simply prepends the global $font-path when appropriate.

Since 4.0.0:

Supports Base64 data URI’s, without prepending font-path.

Parameters & Return

$path: (string)

A path relative to the fonts directory.

@return (string)

The full path to a font file, in CSS url('my/path') format.

Example

scss 
$font-path: '../fonts/';
$src: font-url('sans/my-sans-font.woff2');
/* #{$src} */
css compiled 
/* url("../fonts/sans/my-sans-font.woff2") */

@function font-family()

Access a font-family from your $fonts map, with name and font-stack.

Since 4.0.0:

Provides the correct name for fonts when an alias is given.

Parameters & Return

$key: (string)

The key-name of a font in your $fonts map.

@return (String)

The full font-stack for the given font, ready for output with the font-family property.

Example

scss 
$fonts: (
  'sans': (
    'name': 'FranklinGothic',
    'stack': (Helvetica, Arial, sans-serif),
  ),
);

html {
  font-family: font-family('sans');
}
css compiled 
html {
  font-family: "FranklinGothic", Helvetica, Arial, sans-serif;
}

@mixin font-family()

Output a font-stack to font-family based on your $fonts configuration.

Parameters

$key: (string)

The key-name of a font in your $fonts map.

Example

scss 
$fonts: (
  'FranklinGothic': (
    'stack': (Helvetica, Arial, sans-serif),
  ),
  'sans': 'FranklinGothic',
);

html {
  @include font-family('sans');
}
css compiled 
html {
  font-family: "FranklinGothic", Helvetica, Arial, sans-serif;
}

@mixin font-face()

Import any local webfont defined in your $fonts configuration (variants without a path will not be imported).

Since 4.0.0:

No longer accepts the $formats parameter.

Parameters & Output

$key: (string)

The key-name of a font in your $fonts map.

{CSS output} (code block)

@font-face import blocks for any defined font-variants.

Example

scss 
$fonts: (
  'body-font': (
    'name': 'Source Sans Pro',
    'formats': 'woff2' 'woff' 'eot',
    'normal': 'sans/sourcesanspro-regular-webfont',
    'italic': 'sans/sourcesanspro-italic-webfont',
    'bold': 'sans/sourcesanspro-bold-webfont',
  ),
);
@include font-face('body-font');
css compiled 
@font-face {
  font-family: "Source Sans Pro";
  font-style: normal;
  font-weight: normal;
  src: url("sans/sourcesanspro-regular-webfont.eot");
  src: url("sans/sourcesanspro-regular-webfont.eot?#iefix") format("embedded-opentype"), url("sans/sourcesanspro-regular-webfont.woff2") format("woff2"), url("sans/sourcesanspro-regular-webfont.woff") format("woff");
}

@font-face {
  font-family: "Source Sans Pro";
  font-style: italic;
  font-weight: normal;
  src: url("sans/sourcesanspro-italic-webfont.eot");
  src: url("sans/sourcesanspro-italic-webfont.eot?#iefix") format("embedded-opentype"), url("sans/sourcesanspro-italic-webfont.woff2") format("woff2"), url("sans/sourcesanspro-italic-webfont.woff") format("woff");
}

@font-face {
  font-family: "Source Sans Pro";
  font-style: normal;
  font-weight: bold;
  src: url("sans/sourcesanspro-bold-webfont.eot");
  src: url("sans/sourcesanspro-bold-webfont.eot?#iefix") format("embedded-opentype"), url("sans/sourcesanspro-bold-webfont.woff2") format("woff2"), url("sans/sourcesanspro-bold-webfont.woff") format("woff");
}

@mixin import-webfonts()

@mixin import-webfonts()

Find all the fonts that include data for imports, and generate @font-face blocks for each individual font and variant.

Since 4.0.0:

No longer accepts the $formats parameter.

Parameters & Output

$fonts: $fonts (map)

A map of your font configuration.

{CSS output} (code block)

Any number of @font-face blocks.

Example

scss 
$fonts: (
  'body-font': (
    'name': 'Source Sans Pro',
    'formats': 'woff2' 'woff' 'eot',
    'normal': 'sans/sourcesanspro-regular-webfont',
    'italic': 'sans/sourcesanspro-italic-webfont',
    'bold': 'sans/sourcesanspro-bold-webfont',
  ),
  // alias keys, and fonts without path data will be ignored…
  'alias': 'body-font',
  'google-font': (
    'source': 'http://fonts.google.com/',
  ),
);
@include import-webfonts;
css compiled 
@font-face {
  font-family: "Source Sans Pro";
  font-style: normal;
  font-weight: normal;
  src: url("sans/sourcesanspro-regular-webfont.eot");
  src: url("sans/sourcesanspro-regular-webfont.eot?#iefix") format("embedded-opentype"), url("sans/sourcesanspro-regular-webfont.woff2") format("woff2"), url("sans/sourcesanspro-regular-webfont.woff") format("woff");
}
